## Setup and Installation ðŸ› 

1. Clone the repository

```bash
git clone
```

2. Install dependencies

```bash
npm install
```

3. Run the server

- 3.1 Development mode

```bash
npm run dev
```

- 3.2 Production mode

```bash
npm run build
npm run start
```
